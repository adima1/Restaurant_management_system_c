# Restaurant Management System (C)

## Overview

This project is an extended version of a **Restaurant Management System** written in C.
The original system was taken from an open-source GitHub repository and extended as part of an academic project.

The main goal was to analyze an existing C-based system and add new features while preserving its original structure.

## Features

- Admin and Customer management sections
- Menu and order management using doubly linked lists
- Dynamic memory allocation
- File-based membership system
- Discounts and promotional offers
- Final bill calculation

---

## Added Extensions

The following features were added as part of the academic extension:

- Partial deletion of ordered items
- Membership system with 10% discount
- Special gift for orders above a defined amount
- Free dessert promotion
- Drinks with optional ice selection

Detailed feature descriptions are available in the attached documentation.
